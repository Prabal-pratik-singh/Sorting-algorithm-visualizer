# Sorting Algorithm Visualizer

A web application that visualizes sorting algorithms through animated bars.

## Features
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Real-time visualization
- Adjustable array size and speed

## Technologies Used
- JavaScript
- HTML
- CSS
- Webpack
